aggregators

A collection of comms_ml “aggregation functions” - functions that return a post-processing function. This post-processing converts a list of pre-processed values such as packet timestamp in milliseconds into a fixed-length feature such as the mean.

comms_ml.aggregators.count()

Returns an aggregator for the number of occurrences

comms_ml.aggregators.count_if(condition)

Returns an aggregator for the number of occurrences that match the given condition

comms_ml.aggregators.downsample(factor)

Returns an aggregator for the first valid element of the samples, downsampled with the given factor.

comms_ml.aggregators.head(n_elems)

Returns an aggregator for the first n_elems elements of the first sample

comms_ml.aggregators.iat()

Returns a property map that contains the inter-arrival times of the given packets

comms_ml.aggregators.keep_if(condition)

Returns a property map that keeps only the first element

comms_ml.aggregators.n_unique()

Returns the number of unique elements

comms_ml.aggregators.tail(n_elems)

Returns an aggregator for the last n_elems elements of the first sample
